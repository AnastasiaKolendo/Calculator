# Calculator

## Description

A calculator program that allows a user to enter a math problem as a string and get an answer.
> Test Assignment for VidMob. Technologies used: ES6, Chai.\

**The calculator allows basic arithmetic operations:**

`+` - addition\
`-` - subtraction\
`*` - multiplication\
`รท` - division\

## Install
npm i

## Usage
node calculator.js

## Run tests
npm test
